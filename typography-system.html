Used directly after headings. Slightly larger for visual breathing room before body text.
Body Text 1.00rem (16px) · lh 1.625

The Medallion Architecture (Bronze → Silver → Gold) is a data design pattern used in Databricks Lakehouse to organize data quality layers. Raw ingested data sits in Bronze, cleaned and enriched data in Silver, and business-ready aggregates in Gold. This pattern ensures ACID compliance via Delta Lake.

Core reading size. 16px × 1.625 line-height = 26px line spacing — optimal vertical rhythm.

§ 7 · Performance & Accessibility

Built for Core Web Vitals

Every decision is measured against CLS, FCP, and LCP. WCAG AA minimum, AAA where possible.

⚡
Font Loading — Zero CLS
Preconnect + preload + font-display:swap eliminates layout shift. Size-adjust property matches fallback metrics to loaded font precisely.
@font-face {
font-display: swap;
size-adjust: 100.6%;
ascent-override: 90%;
}
♿
WCAG AA / AAA Compliance
Body text (#0f172a on #fff) = 17.7:1 contrast. Muted text (#475569 on #fff) = 5.9:1 — meets AA. Min body size 15px — no tiny text.

§ 9 · Line Height & Vertical Rhythm

Breathing Room

Line height creates vertical rhythm. The rule: tighter for large type (more visual weight), looser for small type (more air for legibility).
